# Midland Studio

Marketing website for **Midland Studio** — a web design studio serving small businesses across the Richmond to Charlottesville area of Central Virginia.

We build custom websites, local SEO, and practical AI tools for small businesses.

## About this site

A fast, mobile-friendly, fully static website (hand-written HTML + CSS, no framework or build step). Deployed continuously: every push to `main` publishes to production.

### Pages
- `index.html` — home
- `pricing.html` — plans and pricing
- `seo.html` — SEO services
- `ai-services.html` — AI solutions
- `pricing-infographic.html` — shareable pricing graphic
- `social-media-infographic.html` — social media packages graphic
- `social-media-house-rules.html` — client onboarding one-pager
- `seo-guide.html` / `seo-checklist.html` — SEO learning resources

### Brand
- **Cardinal Red** `#c8102e` — primary / calls to action
- **Antique Gold** `#c9a227` — accents
- **Charcoal** `#1a1a1a` — text and dark sections
- **White** — space
